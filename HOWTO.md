



## Neovim

Open netrw (file explorer) by launching `nvim` with no arguments.

Navigate with arrow keys or `hjkl`, enter directories with `Enter`, and open files with `Enter`.

## Tmux

Tmux is a ...

List current sessions:

`tmux ls` (alias: `tl`)

Create a new tmux session with a name:

`tmux new -s <session_name>`

Install plugins with ctrl+b + ctrl+I

Reload config with ctrl+b + ctrl+r
